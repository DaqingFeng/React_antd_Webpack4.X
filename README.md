# Antd for New template

## Usage

```bash
$ git clonehttps://github.com/DaqingFeng/React_antd_Webpack4.x.git --depth=1
$ cd React_antd_Webpack4.x
$ npm install
$ npm start         # visit http://localhost:8100
```
