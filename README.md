# Antd for webpack 4.x

## Usage

```bash
$ git clone https://github.com/DaqingFeng/antd_webpack4.x.git --depth=1
$ cd React_antd_Webpack4.x
$ npm install
$ npm start         # visit http://localhost:8100
```
